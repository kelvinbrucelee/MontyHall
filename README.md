# Monty Hall

# Getting Started

>**Note**: Project created in [NextJs](https://nextjs.org/docs) instructions till "Creating a new application" step, before proceeding.

## Step 1: Install Node Modules

```bash
# using npm
npm install node
```

## Step 2: Start Project

```bash
# using npm
npm run dev
```
